# Codentra - Software Company Project Marketplace

🚀 **منصة متكاملة لبيع وشراء المشاريع البرمجية**

---

## 📋 نظرة عامة

**Codentra** هو سوق إلكتروني متخصص للمشاريع البرمجية، يربط بين المطورين والعملاء الذين يبحثون عن حلول جاهزة. المنصة توفر نظاماً كاملاً لإدارة المشاريع، المعاملات المالية، الفواتير، والاشتراكات.

---

## 🏗️ بنية المشروع

```
codentra/
├── 📱 mobile-app/           # تطبيق الموبايل (React Native + Expo)
├── 🌐 server.js             # الخادم الخلفي (Node.js + Express)
├── 📁 views/                # واجهات الويب (EJS Templates)
├── 📁 public/               # ملفات静态ة (CSS, JS, Images)
├── 📁 data/                 # قاعدة البيانات (ملفات JSON)
├── 📁 uploads/              # ملفات المرفوعات
└── 📁 node_modules/         # الحزم Dependencies
```

---

## 🛠️ التقنيات المستخدمة

### الخادم الخلفي (Backend)
- **Node.js** - بيئة التشغيل
- **Express.js** - إطار العمل الويب
- **Socket.io** - الاتصالات الفورية
- **JWT** - مصادقة المستخدمين
- **Multer** - رفع الملفات
- **PDFKit** - إنشاء الفواتير PDF
- **bcryptjs** - تشفير كلمات المرور
- **EJS** - محرك القوالب

### تطبيق الموبايل (Mobile App)
- **React Native** - إطار العمل
- **Expo** - منصة التطوير
- **React Navigation** - التنقل بين الشاشات
- **Expo Secure Store** - تخزين آمن
- **React Native Safe Area Context** - مناطق آمنة

### قاعدة البيانات
- **ملفات JSON** - تخزين البيانات (بسيط وسريع)

## 🚀 طريقة التشغيل

### المتطلبات الأساسية
```bash
# تثبيت Node.js (الإصدار 16 أو أحدث)
node --version

# تثبيت npm
npm --version

# تثبيت Expo CLI (للموبايل)
npm install -g expo-cli
```

### تشغيل الخادم الخلفي
```bash
# الدخول لمجلد المشروع
cd /Users/ahmedsamir/Desktop/codentra

# تثبيت الحزم
npm install

# تشغيل الخادم
npm start
# أو
npm run dev  # للطور التطويري
```

### تشغيل تطبيق الموبايل
```bash
# الدخول لمجلد الموبايل
cd mobile-app

# تثبيت الحزم
npm install

# تشغيل التطبيق
expo start

# تشغيل على أجهزة محددة
expo start --android    # للأندرويد
expo start --ios        # للأيفون
expo start --web        # للويب
```

## 🌐 إعدادات الشبكة

### تكوين الـ IP للتطبيق
في ملف `mobile-app/config/api.js`:

```javascript
export const API_CONFIG = {
  BASE_URL: 'http://192.168.8.101:3000',  // IP الشبكة الحالية
  TIMEOUT: 30000,
  MAX_RETRIES: 5,
  RETRY_DELAY: 2000,
};
```

### تغيير الـ IP
```javascript
// اختر الـ IP المناسب لشبكتك:

};
```

## 🚀 Deployment

### للخادم الخلفي
```bash
# استخدام PM2 للإنتاج
npm install -g pm2
pm2 start server.js --name "codentra-api"

# أو باستخدام Docker
docker build -t codentra-api .
docker run -p 3000:3000 codentra-api
```

### لتطبيق الموبايل
```bash
# بناء للأندرويد
expo build:android

# بناء للأيفون
expo build:ios

# بناء للويب
expo build:web
```

## 🏢 عن Codentra
